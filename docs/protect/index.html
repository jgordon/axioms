Predicate: protect

Additional Axioms

The function of a building is to protect entities inside it.

(forall (x)
  (if (building x)
      (exists (f y)
        (and (function f x) (protect' f x y)))))

(forall (f x y)
  (if (and (function f x) (protect' f x y) (etc f x y))
      (building x)))
