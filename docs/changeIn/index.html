Predicate: changeIn

Summary

(changeIn x)

There is a change in some property of x.

Characterizing Axioms

(forall (e x)
  (iff (changeIn' e x)
       (exists (e1 e2 e3)
         (and (arg* x e1) (arg* x e2) (change' e3 e1 e2) (gen e3 e)))))
