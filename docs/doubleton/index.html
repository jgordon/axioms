Predicate: doubleton

Summary

(doubleton s x y)

s consists of the two elements x and y.

Characterizing Axioms

(forall (s x y)
  (iff (doubleton s x y)
       (exists (s1)
         (and (nequal x y)
              (addElt s s1 x)
              (singleton s1 y)))))

Additional Axioms

(forall (x y z)
  (iff (aggregate x y z)
       (and (compositeEntity x)
            (exists (s)
              (and (componentsOf s x) (doubleton s y z))))))
