Predicate: nequal

Summary

(nequal x y)

x is not equal to y.

Characterizing Axioms

(forall (x y)
  (iff (nequal x y)
       (not (equal x y))))

Additional Axioms

(forall (s x y)
  (iff (doubleton s x y)
       (exists (s1)
         (and (nequal x y)
              (addElt s s1 x)
              (singleton s1 y)))))
