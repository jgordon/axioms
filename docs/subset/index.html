Predicate: subset

Summary

(subset s1 s2)

s1 is a subset of s2.

Characterizing Axioms

Subsets and proper subsets:

(forall (s1 s2)
  (iff (subset s1 s2)
       (and (set s1) (set s2)
            (forall (x)
              (if (member x s1)
                  (member x s2))))))

Additional Axioms

(forall (s1 s2)
  (iff (properSubset s1 s2)
       (and (subset s1 s2) (nequal s1 s2))))
