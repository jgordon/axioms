Predicate: vehicleAnchoredFramework

Characterizing Axioms

A vehicle-anchored framework is a self-anchored framework where the self is a boat or airplane.

(forall (f v)
   (iff (vehicleAnchoredFramework f v)
        (and (selfAnchoredFramework f v)
             (or (boat v) (airplane v)))))
