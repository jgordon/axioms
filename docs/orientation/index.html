Predicate: orientation

Additional Axioms

Physical objects have a front have an intrinsic orientation, and that orientation can change. This is called turning.

(forall (r x)
   (if (front r x)
       (iff (turn x d1 d2)
            (exists (e1 e2 f)
               (and (change e1 e2) (orientation' e1 d1 x f)
                    (orientation' e2 d2 x f))))))
