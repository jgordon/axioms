Predicate: decrease

Summary

(decrease x s)

x decreases on scale s.

Characterizing Axioms

(forall (e x y z s)
  (iff (decrease x s)
       (exists (y z e1 e2)
         (and (at' e1 x y s) (at' e2 x z s) (lts z y s)
              (vertical s) (change e1 e2)))))
