Predicate: vertical

Summary

(vertical s)

Scale s is a real or metaphorical vertical scale.

Characterizing Axioms

The argument of vertical has to be a scale.

(forall (s) (if (vertical s) (scale s)))

Additional Axioms

A non-negative numeric scale is vertical.

(forall (s) (if (nonNegNumericScale s) (vertical s)))
