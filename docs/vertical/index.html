Predicate: vertical

Summary

(vertical s)

Scale s is a real or metaphorical vertical scale.

Characterizing Axioms

The argument of vertical has to be a scale.

(forall (s) (if (vertical s) (scale s)))

All Axioms

A non-negative numeric scale is vertical.

(forall (s) (if (nonNegNumericScale s) (vertical s)))

The argument of vertical has to be a scale.

(forall (s) (if (vertical s) (scale s)))

If an entity moves from a point on vertical scale to a higher point, we say that there as been an increase.

(forall (e x y z s)
  (iff (increase x s)
       (exists (y z e1 e2)
         (and (at' e1 x y s) (at' e2 x z s) (lts y z s)
              (vertical s) (change e1 e2)))))

Similarly, if an entity moves from a point on a vertical scale to a lower point, there has been a decrease.

(forall (e x y z s)
  (iff (decrease x s)
       (exists (y z e1 e2)
         (and (at' e1 x y s) (at' e2 x z s) (lts z y s)
              (vertical s) (change e1 e2)))))

The likelihood scale is conceptually vertical. We talk about likelihoods being higher or lower.

(forall (s) (if (likelihoodScale s) (vertical s)))
