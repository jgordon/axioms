Predicate: fraction

Summary

(fraction f a b)

f is the fraction whose numerator is a and whose denominator is b; f = a/b.

Characterizing Axioms

(forall (f a b)
  (if (fraction f a b) (number f)))

(forall (f a b)
  (if (fraction f a b)
      (and (number a) (number b) (nequal b 0))))

Additional Axioms

(forall (a b)
   (if (and (nonNegInteger a) (posInteger b))
       (exists (f) (fraction f a b))))
