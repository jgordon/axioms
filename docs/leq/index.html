Predicate: leq

Summary

(leq n1 n2)

n1 is less than or equal to n2.

Characterizing Axioms

(forall (n1 n2)
  (iff (leq n1 n2)
       (or (lt n1 n2) (equal n1 n2))))

Additional Axioms

(forall (e)
  (iff (eventuality e)
       (exists (p n)
         (and (pred p e) (arity n e)
              (forall (i)
                (if (and (posInteger i)
                         (leq i n))
                    (exists (x)
                      (argn x i e))))))))

(forall (x e1 y e2)
  (iff (subst x e1 y e2)
       (exists (p n)
         (and (pred p e1) (pred p e2) (arity n e1) (arity n e2)
              (noSubstTypelt x e1 y e2)
              (forall (i z1 z2)
                (if (and (posInteger i) (leq i n)
                         (argn z1 i e1) (argn z2 i e2))
                    (and (iff (eventuality z1)
                              (eventuality z2))
                         (if (not (eventuality z1))
                             (and (if (nequal z1 x)
                                      (equal z2 z1))
                                  (if (equal z1 x)
                                      (equal z2 y))))
                         (if (eventuality z1)
                             (subst x z1 y z2)))))))))

(forall (s n1 n2)
  (iff (ints s n1 n2)
       (and (posInteger n1) (posInteger n2) (set s)
            (forall (n)
              (iff (member n s)
                   (and (posInteger n) (leq n1 n) (leq n n2)))))))

Are two entites the same half order of magnitude? (Is the proportion of the smaller to the larger less than the square root of 10?)

(forall (x y s1 m)
  (if (and (leqs x y s1) (measure m s1))
      (iff (sameHOM x y s1 m)
           (exists (f f2)
             (and (proportion f y x m) (product f2 f f)
                  (leq f2 10))))))

The triangle inequality holds for distance.

(forall (d1 d2 d3 d4 x1 x2 x3 u)
  (if (and (distance d1 x1 x2 u) (distance d2 x2 x3 u)
           (distance d3 x1 x3 u) (sum d4 d1 d2))
      (leq d3 d4)))

For a given bounding box we can define the primary, secondary, and tertiary dimensions of the physical object. In this axiom c1, c2, and c3 are three non-parallel sides in decreasing order of length, and d1, d2 and d3 are their respective lengths, measured in units u. The predicate “dimi” picks out the ith longest side.

(forall (c1 c2 c3 d1 d2 d3 u b x)
   (if (and (boundingBox b x) (edge c1 b) (edge c2 b) (edge c3 b)
            (not (parallel c1 c2)) (not (parallel c2 c3))
            (not (parallel c1 c3))
            (length d1 c1 u) (length d2 c2 u) (length d3 c3 u))
       (and (iff (dim1 c1 b x)
                 (and (lt d2 d1) (lt d3 d1)))
            (iff (dim2 c2 b x)
                 (and (leq d2 d1) (leq d3 d2)))
            (iff (dim3 c3 b x)
                 (and (lt d3 d1) (lt d3 d2))))))
