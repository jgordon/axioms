Predicate: concave

Additional Axioms

A planar region is concave if it is not convex.

(forall (r)
  (if (and (region r) (planar r))
      (iff (concave r) (not (convex r)))))

A volume is concave if it is not convex.

(forall (p)
   (if (volume p)
       (iff (concave p) (not (convex p)))))
