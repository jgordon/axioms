Predicate: possible

(possible e c)

e is possible with respect to a set of constraints c.

Characterizing Axioms

(forall (e c)
  (if (possible e c)
      (PosMod e possible)))


(forall (e c)
  (if (possible e c)
      (and (eventuality e) (eventualities c))))


(forall (e c)
  (iff (possible e c)
       (forall (e1)
         (if (not' e1 e)
             (not (imply c e1))))))
