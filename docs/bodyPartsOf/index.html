Predicate: bodyPartsOf

Summary

(bodyPartsOf s p)

s is the set of p's body parts.

Characterizing Axioms

A person p has a set of body parts, namely, the components of the body.

(forall (p s)
  (iff (bodyPartsOf s p)
       (exists (b)
         (and (person p) (body b p) (componentsOf s b)))))

Additional Axioms

The sense organs are a subset of the body parts.

(forall (p x)
  (if (and (person p) (bodyPartsOf s1 p))
      (exists (s2)
        (and (subset s2 s1)
             (forall (o)
               (iff (member o s2) (senseOrgan o p)))))))
