Predicate: sameDuration

Summary

(sameDuration t1 t2)

Temporal entities t1 and t2 have the same duration.

Characterizing Axioms

(forall (t1 t2)
  (if (sameDuration t1 t2)
      (and (temporalEntity t1) (temporalEntity t2))))

(forall (t1 t2)
  (iff (sameDuration t1 t2) (sameDuration t2 t1)))
