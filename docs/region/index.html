Predicate: region

Summary

(region r)

r is a region, a 2-dimensional entity.

Characterizing Axioms

A region is a figure.

(forall (r) (if (region r) (figure r)))

Additional Axioms

A (2-dimensional) region can have a border, which is a (1-dimensional) curve.

(forall (c r)
  (if (border c r)
      (and (curve c) (region r))))
