Predicate: replaceElt

Summary

(replaceElt s s1 x y)

s is obtained from s1 by replacing x with y.

Characterizing Axioms

The predication (replaceElt s s1 x y) says that set s is obtained from set s1 by replacing element x with element y.

(forall (s s1 x y)
  (iff (replaceElt s s1 x y)
       (exists (s2)
         (and (deleteElt s2 s1 x)
              (addElt s s2 y)))))
