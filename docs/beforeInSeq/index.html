Predicate: beforeInSeq

Summary

(beforeInSeq x y s)

x comes before y in the sequence s.

Characterizing Axioms

(forall (x y s)
  (iff (beforeInSeq x y s)
       (exists (n1 n2)
         (and (sequence s) (nth x n1 s) (nth y n2 s)
              (lt n1 n2)))))
