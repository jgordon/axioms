Predicate: disjoint

Summary

(disjoint s1 s2)

The component sets of scales s1 and s2 are disjoint.

All Axioms

(forall (m s1 s2 x y)
  (if (and (measure m s1) (function m s1 s2) (disjoint s1 s2)
           (map m x y))
      (at x y s2)))

(forall (s1 s2)
