Predicate: disjoint

Summary

(disjoint s1 s2)

The component sets of scales s1 and s2 are disjoint.

Additional Axioms

(forall (s1 s2)
  (if (and (set s1) (set s2))
      (iff (disjoint s1 s2)
           (not (exists (x)
                  (and (member x s1) (member x s2)))))))
