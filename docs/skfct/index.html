Predicate: skfct

Summary

(skfct y1 y x1 x e)

y1 is the value of the function corresponding to the functional dependency of y on x with respect to e when applied to x1.

Characterizing Axioms

(forall (x x1 y y1 e)
  (iff (skfct y1 y x1 x e)
       (and (fd y x e)
            (exists (e1)
              (and (subst2 x y e x1 y1 e1)
                   (if (Rexist e) (Rexist e1)))))))
