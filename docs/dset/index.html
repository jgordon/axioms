Predicate: dset

Summary

(dset s x e)

s is the set with typical element x defined by the property e.

Characterizing Axioms

Real members of a set inherit all the properties of the typical element.

(forall (s x e)
  (if (dset s x e)
      (and (set s) (typelt x s) (arg* x e))))
