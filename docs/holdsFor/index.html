Predicate: holdsFor

Summary

(holdsFor e1 y x)

The eventuality type e1 holds for entity y; substituting y for a nonspecific argument x of e1 yields an eventuality that really exists.

Characterizing Axioms

(forall (e1 y x)
  (iff (holdsFor e1 y x)
       (exists (s e2)
         (and (subst y e2 x e1) (typelt x s) (member y s)
              (Rexist e2)))))
