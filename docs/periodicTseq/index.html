Summary

(periodicTseq s)

Temporal sequence s is periodic, in that the gaps between successive elements are all equal.

Characterizing Axioms

(forall (s)
  (iff (periodicTseq s)
       (and (tseq s)
            (exists (d u)
              (forall (t1 t2)
                (if (successiveElts t1 t2 s)
                    (exists (t)
                      (and (intervalBetween t t1 t2)
                           (durationOf d t u)))))))))
