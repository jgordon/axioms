Predicate: Hi

Summary

(Hi s1 s)

s1 is the high region of scale s.

Characterizing Axioms

We can construct composite scales out of existing scales.

(forall (s1 s) (if (Hi s1 s) (subscale s1 s)))

The top of the scale is in the Hi region of the scale and is the top of the Hi region.

(forall (s1 s)
   (if (Hi s1 s)
       (forall (x)
         (iff (top x s) (top x s1)))))

Additional Axioms

If a point is in the Lo region, then it is less than all the points in the Hi region.

(forall (s s1 s2 x y)
   (if (and (Hi s1 s) (Lo s2 s) (inScale x s1) (inScale y s2))
       (lts y x s)))

The scaleFor predicate relates a scale and the absolute form of its adjective (e.g., 'tall').

(forall (s e)
   (iff (scaleFor s e)
        (exists (s1)
           (and (Hi s1 s)
                (forall (e1 x y)
                   (if (and (at' e1 x y s1) (argn x 1 e))
                       (iff (Rexist e) (Rexist e1))))))))

If something is in the Hi region of a scale, then that property plays a causal or enabling role in some agent's goal being achieved or not achieved.

(forall (e x y s1 s)
   (if (and (at' e x y s1) (Hi s1 s) (etc))
       (exists (c a g g1)
          (and (member e c) (goal g a)
               (or (causalComplex c g)
                   (and (not' g1 g) (causalComplex c g1)))))))

A set of events is frequent if it is in the Hi region of a rate scale.

(forall (s1)
  (iff (frequent s1)
       (exists (s s0 s2)
         (and (member s1 s0) (rateScale s s0) (Hi s2 s)
              (inScale s1 s2)))))

(forall (x z)
   (iff (far x z)
        (exists (s s2)
           (and (nearnessScale s z) (Hi s2 s) (onScale x s2)))))

(forall (x s0 s1)
   (if (and (onScale x s1) (Hi s1 s0) (etc))
       (exists (s2 s3 s4 n3 n4)
          (and (componentsOf s2 s0) (union s2 s3 s4)
               (forall (y) (iff (member y s3) (lts y x s0)))
               (forall (z) (iff (member z s4) (lts x z s0)))
               (card n3 s3) (card n4 s4) (lt n4 n3)))))

An eventuality is likely with respect to constraints c if its likelihood d is on the high end of the scale of likelihoods.

(forall (e c)
  (iff (likely e c)
       (exists (s d s1)
         (and (likelihood d e c) (likelihoodScale s) (Hi s1 s)
              (inScale d s1)))))
