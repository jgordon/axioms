Predicate: temporal

Summary

(temporal e)

e is a temporal property, i.e., `atTime`, `during`, `timeSpanOf`, `happensIn`, `before`, or `durationOf`.

All Axioms

(forall (e) (iff (nontemporal e) (not (temporal e))))
