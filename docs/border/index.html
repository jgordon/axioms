Predicate: border

Summary

(border c r)

Curve c is the border of region r.

Characterizing Axioms

(forall (c r)
  (if (border c r)
      (and (curve c) (region r))))

All Axioms

(forall (c r)
  (if (border c r)
      (and (curve c) (region r))))

(forall (f1 f2)
