Predicate: monotoneIncreasing

Summary

(monotoneIncreasing f)

Function f is monotone-increasing scale-to-scale function preserving the scales' “less than” ordering.

Additional Axioms

(forall (f s1 s2)
   (if (and (function f s1 s2) (scale s1) (scale s2))
       (iff (monotoneIncreasing f)
            (forall (x1 x2 y1 y2)
               (if (and (map f x1 y1) (map f x2 y2) (lts x1 x2 s1))
                   (lts y1 y2 s2))))))
