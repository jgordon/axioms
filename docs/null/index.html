Predicate: null

Summary

(null s)

s is the null set or empty set.

Characterizing Axioms

(forall (s)
  (iff (null s)
       (and (set s)
            (forall (x)
              (not (member x s))))))

(forall (s u)
  (if (null s) (durationOf 0 s u)))

Additional Axioms

(forall (s x)
  (iff (singleton s x)
       (exists (s1)
         (and (addElt s s1 x) (null s1)))))
