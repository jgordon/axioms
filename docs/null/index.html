Predicate: null

Summary

(null s)

s is the null set or empty set.

Characterizing Axioms

(forall (s)
  (iff (null s)
       (and (set s)
            (forall (x)
              (not (member x s))))))

A temporal entity may be an instant or an interval.

(forall (s u)
  (if (null s) (durationOf 0 s u)))

Additional Axioms

(forall (s x)
  (iff (singleton s x)
       (exists (s1)
         (and (addElt s s1 x) (null s1)))))

(forall (n s)
  (iff (card n s)
       (and (nonNegInteger n) (set s)
            (or (and (null s) (equal n 0))
                (exists (s1 x m)
                  (and (addElt s s1 x)
                       (not (member x s1))
                       (card m s1)
                       (sum n m 1)))))))

A defined set consists only of elements for which a particular eventuality really exists.

(forall (e1 e)
  (iff (instance e1 e)
       (and (partialInstance e1 e)
            (forall (s1)
              (if (parameters s1 e1)
                  (null s1))))))

(forall (s1 x)
  (if (componentsOf s1 x)
      (and (set s1) (compositeEntity x) (not (null s1)))))

(forall (e s s1 s2 s3)
  (if (and (set' e s) (Rexist e) (not (null s))
           (singleton s2 e) (null s3))
      (and (compositeEntity s) (componentsOf s s)
           (propertiesOf s2 s) (relationsOf s3 s))))

(forall (s1 e x y)
  (if (and (forall (s2)
             (if (member s2 s1)
                 (set s2)))
            (subset' e x y))
       (exists (s)
          (and (scaleDefinedBy s s1 e)
               (forall (z)
                 (if (and (member z s1) (null z))
                     (bottom z s)))))))

A non-negative integer n1 is less than n2 iff there is some positive integer n3 s.t. n2 is the sum of n1 and n3.

(forall (e x y s1 s m n u)
  (if (and (forall (z)
             (if (member z s1) (set z)))
           (exists (w)
             (and (member w s1) (null w)))
           (subset' e x y)
           (scaleDefinedBy s s1 e)
           (card' m n u))
      (measure m s)))

A non-null subset of the domain is horizontal if no element is above2 any other element. We use the predicate horizontal2 because as stated above, the two- and three-dimensional cases differ. The next two axioms refer to the above3 relation. In the two-dimensional case, this is vacuous since nothing is above3 anything else. But it will greatly simplify our treatment of three-dimensional frameworks to include it here.

(forall (s1 f)
  (iff (horizontal2 s1 f)
       (exists (s)
         (and (domain s f) (subset s1 s) (not (null s1))
              (not (exists (x1 x2)
                     (and (member x1 s1) (member x2 s1)
                          (or (above2 x1 x2 f)
                              (above3 x1 x2 f)))))))))

A similar axiom defines vertical2 in terms of rightOf.

(forall (s1 f)
  (iff (vertical2 s1 f)
       (exists (s)
         (and (domain s f) (subset s1 s) (not (null s1))
              (not (exists (x1 x2)
                     (and (member x1 s1) (member x2 s1)
                          (or (rightOf x1 x2 f)
                              (above3 x1 x2 f)))))))))

A y-coordinate can be defined similarly.

(forall (s1 f)
   (iff (vertical3 s1 f)
        (exists (s)
           (and (domain s f) (subset s1 s) (not (null s1))
                (not (exists (x1 x2)
                        (and (member x1 s1) (member x2 s1)
                             (or (rightOf x1 x2 f)
                                 (above2 x1 x2 f)))))))))
