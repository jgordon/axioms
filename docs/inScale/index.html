Predicate: inScale

Summary

(inScale y s)

y is a component of the scale s.

Characterizing Axioms

(forall (y s)
   (iff (inScale y s)
        (and (scale s) (componentOf y s))))

Additional Axioms

We can apply lts and gts either to points on the scale or to entities that are at those points.

(forall (x y s)
   (if (lts x y s)
       (and (or (inScale x s)
                (exists (x1) (and (at x x1 s) (inScale x1 s))))
            (or (inScale y s)
                (exists (y1) (and (at y y1 s) (inScale y1 s)))))))
