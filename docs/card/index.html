Predicate: card

Summary

(card n s)

n is the cardinality of s.

Characterizing Axioms

(forall (n s)
  (iff (card n s)
       (and (nonNegInteger n) (set s)
            (or (and (null s) (equal n 0))
                (exists (s1 x m)
                  (and (addElt s s1 x)
                       (not (member x s1))
                       (card m s1)
                       (sum n m 1)))))))
