Predicate: notSubsetTypelt

Summary

(notSubsetTypelt y s)

y is not a typical element of a subset of s; this blocks problematic applications of Axiom 6.

Characterizing Axioms

"Surgery" on the definition of "dset" to rule out problematic cases:

(forall (y s)
  (iff (notSubsetTypelt y s)
       (not (exists (s1)
              (and (typelt y s1) (subset s1 s))))))

All Axioms

Real members of a set inherit all the properties of the typical element.

(forall (s x e y e1)
  (if (and (dset s x e) (subst x e y e1) (Rexist e)
           (notSubsetTypelt y s))
      (iff (member y s) (Rexist e1))))
