Predicate: leqs

Summary

(leqs x y s)

x is less than or equal to y on scale s.

Characterizing Axioms

(forall (x y s)
   (iff (leqs x y s)
        (or (lts x y s)
            (and (onScale x s) (equal x y)))))

Additional Axioms

(forall (x y s)
   (if (and (leqs x y s) (leqs y x s))
       (equal x y)))
