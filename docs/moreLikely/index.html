Predicate: moreLikely

Summary

(moreLikely e1 e2 c)

e1 is more likely than e2 given constraints c.

Characterizing Axioms

(forall (e1 e2 c)
  (iff (moreLikely e1 e2 c)
       (exists (s d1 d2)
         (and (likelihoodScale s)
              (likelihood d1 e1 c) (likelihood d2 e2 c)
              (lts d2 d1 s)))))
