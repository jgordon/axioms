Predicate: eventualities

Summary

(eventualities s)

s is a set of eventualities.

Characterizing Axioms

(forall (s)
  (iff (eventualities s)
       (and (set s)
            (forall (e)
              (if (member e s)
                  (eventuality e))))))

Additional Axioms

(forall (e s)
  (if (disjunction e s)
      (and (eventuality e) (eventualities s)
           (iff (Rexist e)
                (exists (e1)
                  (and (member e1 s) (Rexist e1)))))))
