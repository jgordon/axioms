Predicate: componentOrWhole

Summary

(componentOrWhole y x)

y is either a component of x or x itself.

Characterizing Axioms

(forall (x1 x)
  (iff (componentOrWhole x1 x)
       (or (equal x1 x) (componentOf x1 x))))
