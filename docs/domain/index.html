Predicate: domain

Summary

(domain s1 f)

s1 is the domain of function f.

Additional Axioms

A function f from a set s1 onto a set s2 is a set of pairs where each element of s1 occurs exactly once as the first element in a pair, and every element of s2 occurs as a second element of at least one pair. (This is function0 since it is later extended to functions from scales to scales.)

(forall (f s1 s2) (if (function0 f s1 s2) (domain s1 f)))

(forall (f)
  (if (framework f)
      (exists (s)
        (and (set s) (domain s f)))))

(forall (s1 f)
  (iff (horizontal2 s1 f)
       (exists (s)
         (and (domain s f) (subset s1 s) (not (null s1))
              (not (exists (x1 x2)
                     (and (member x1 s1) (member x2 s1)
                          (or (above2 x1 x2 f)
                              (above3 x1 x2 f)))))))))

(forall (s1 f)
  (iff (vertical2 s1 f)
       (exists (s)
         (and (domain s f) (subset s1 s) (not (null s1))
              (not (exists (x1 x2)
                     (and (member x1 s1) (member x2 s1)
                          (or (rightOf x1 x2 f)
                              (above3 x1 x2 f)))))))))

(forall (s1 f)
   (iff (vertical3 s1 f)
        (exists (s)
           (and (domain s f) (subset s1 s) (not (null s1))
                (not (exists (x1 x2)
                        (and (member x1 s1) (member x2 s1)
                             (or (rightOf x1 x2 f)
                                 (above2 x1 x2 f)))))))))
