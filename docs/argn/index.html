Predicate: argn

Summary

(argn x n e)

x is the nth argument of e.

Characterizing Axioms

Formulas 1.1 through 1.6 are either meta-rules, listed above, or examples, listed at the end of this section.

(forall (e x n)
  (if (argn x n e)
      (and (nonNegInteger n) (eventuality e))))

Additional Axioms

(forall (e)
  (iff (eventuality e)
       (argn e 0 e)))
