Predicate: ray

Summary

(ray c)

c is a ray.

Characterizing Axioms

A ray has exactly one endpoint.

(forall (c)
  (if (ray c)
      (exists (x)
        (and (endpoint x c)
             (forall (x1)
               (if (endpoint x1 c) (equal x1 x)))))))

Additional Axioms

A line segment has two distinct endpoints.

(forall (c)
  (iff (linear c)
       (or (line c) (ray c) (lineSegment c))))

A direction with respect to a framework is a ray whose endpoint is the origin of the framework.

(forall (d f)
   (iff (direction d f)
        (exists (o)
           (and (ray d) (endpoint o d) (origin o f)))))
