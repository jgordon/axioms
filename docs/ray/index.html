Predicate: ray

(ray c)

c is a ray.

Characterizing Axioms

(forall (c)
  (if (ray c)
      (exists (x)
        (and (endpoint x c)
             (forall (x1)
               (if (endpoint x1 c) (equal x1 x)))))))
