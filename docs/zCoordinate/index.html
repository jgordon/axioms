Predicate: zCoordinate

Characterizing Axioms

We can define the z-coordinate of an element in a framework just as we defined x- and y-coordinates.

(forall (z1 x f)
   (iff (zCoordinate z1 x f)
        (exists (v a1)
           (and (vertical3 v f) (member x v) (zAxis a3 f)
                (member z1 v) (member z1 a3)))))
