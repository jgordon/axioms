Predicate: disjunction

Summary

(disjunction e s)

e is the eventuality of some member of s existing.

Characterizing Axioms

The disjunction of a set really exists exactly when one of the members of the set really exists.

(forall (e s)
  (if (disjunction e s)
      (and (eventuality e) (eventualities s)
           (iff (Rexist e)
                (exists (e1)
                  (and (member e1 s) (Rexist e1)))))))
