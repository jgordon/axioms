Predicate: bodyPart

Summary

(bodyPart x p)

x is one of p's body parts.

All Axioms

(forall (b p)
  (if (body b p)
      (and (spatialSystem b)
           (exists (s)
             (and (componentsOf s b)
                  (forall (x)
                    (iff (member x s) (bodyPart x p))))))))

(forall (o p)
  (if (senseOrgan o p)
      (bodyPart o p)))
