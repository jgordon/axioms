Predicate: rest

Summary

(rest s1 s)

s1 is the rest of the sequence s after the first element of s.

Characterizing Axioms

(forall (s1 s)
  (iff (rest s1 s)
       (and (sequence s) (sequence s1)
            (forall (i x)
              (iff (map s1 i x)
                   (exists (j)
                     (and (posInteger i) (successor j i)
                          (map s j x))))))))

Additional Axioms

(forall (e s e1)
  (iff (forAllOfSeq e s x e1)
       (exists (y e2 l e3 s1)

             (and (length l s) (gt l 1) (first y s) (subst y e2 x e1)
                  (eventSequence e e2 e3) (forAllOfSeq e3 s1 x e1)
                  (rest s1 s))))))
