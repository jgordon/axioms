Predicate: arity

Summary

(arity n e)

n is the arity or the number of arguments of e.

Characterizing Axioms

(forall (n e)
  (if (arity n e)
      (and (nonNegInteger n) (eventuality e))))

Additional Axioms

(forall (e)
  (iff (eventuality e)
       (exists (p n)
         (and (pred p e) (arity n e)
              (forall (i)
                (if (and (posInteger i)
                         (leq i n))
                    (exists (x)
                      (argn x i e))))))))
