Predicate: conflict

Summary

(conflict c x y g h)

c is a conflict between x and y with goals g and h respectively.

Characterizing Axioms

A conflict has two adversaries.

(forall (c x y g h)
  (if (conflict c x y g h)
      (exists (e)
        (adversary e x y c))))

Additional Axioms

In conflict, adversaries attack each other.

(forall (c x y g h e)
  (if (and (conflict c x y g h) (adversary e x y c))
      (attack e x y)))
