Predicate: onScale

Summary

(onScale x s)

x is a point on scale s or at a point on s.

Characterizing Axioms

It is convenient to have a predicate onScale for both points on a scale and for entities at points in a scale.

(forall (x s)
   (iff (onScale x s)
        (or (inScale x s)
            (exists (y)
               (and (inScale y s) (at x y s))))))

All Axioms

Original:

(forall (x y s)
   (if (lts x y s)
       (and (onScale x s) (onScale y s))))
