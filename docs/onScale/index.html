Predicate: onScale

Summary

(onScale x s)

x is a point on scale s or at a point on s.

Characterizing Axioms

(forall (x s)
   (iff (onScale x s)
        (or (inScale x s)
            (exists (y)
               (and (inScale y s) (at x y s))))))

Additional Axioms

Rewritten to use onScale:

(forall (x y s)
   (if (lts x y s)
       (and (onScale x s) (onScale y s))))
