Predicate: set

Summary

(set s)

s is a set.

Characterizing Axioms

(forall (s1 s2)
  (if (set s1)
      (iff (equal s1 s2)
           (and (set s2)
                (forall (x)
                  (iff (member x s1)
                       (member x s2)))))))

(forall (s) (iff (set s)
                 (exists (x) (typelt x s))))

(forall (s)
  (if (set s)
      (iff (Rexist s)
           (forall (e)
             (if (member e s) (Rexist e))))))
