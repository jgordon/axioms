Predicate: sequence

Summary

(sequence s)

s is a sequence.

Characterizing Axioms

n positive integers. There are no constraints on s2.

(forall (s)
  (iff (sequence s)
       (exists (s1 s2 n)
         (and (function0 s s1 s2) (ints s1 1 n)))))

(forall (n s)
  (if (sequence s)
      (iff (length n s)
           (exists (s1 s2)
             (and (function0 s s1 s2) (ints s1 1 n))))))

(forall (s n y)
  (if (sequence s)
      (iff (nth y n s) (map s n y))))

(forall (s y)
  (if (sequence s)
      (iff (first y s) (nth y 1 s))))

Additional Axioms

s after the first is removed.

(forall (s1 s)
  (iff (rest s1 s)
       (and (sequence s) (sequence s1)
            (forall (i x)
              (iff (map s1 i x)
                   (exists (j)
                     (and (posInteger i) (successor j i)
                          (map s j x))))))))
