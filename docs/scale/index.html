Predicate: scale

Summary

(scale s)

s is a scale.

Characterizing Axioms

(forall (s)
   (if (scale s)
       (and (compositeEntity s)
            (exists (s1 e x y s2 s3)
               (and (componentsOf s1 s)
                    (partialOrdering e x y s)
                    (subset s2 s3) (relationsOf s3 s)
                    (forall (e1)
                       (if (member e1 s2)
                           (exists (x1 y1)
                              (and (member x1 s1) (member y1 s1)
                                   (subst2 x1 y1 e1 x y e))))))))))

(forall (s)
  (if (scale s) (ground s)))

Additional Axioms
