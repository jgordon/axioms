Predicate: member

Summary

(member x s)

x is a member of s.

Characterizing Axioms

(forall (x s)
  (if (member x s)
      (set s)))

Additional Axioms

(forall (s1 s2)
  (if (set s1)
      (iff (equal s1 s2)
           (and (set s2)
                (forall (x)
                  (iff (member x s1)
                       (member x s2)))))))
