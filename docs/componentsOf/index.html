Predicate: componentsOf

Summary

(componentsOf s1 x)

s1 is the set of components of composite entity x.

Characterizing Axioms

(forall (s1 x)
  (if (componentsOf s1 x)
      (and (set s1) (compositeEntity x) (not (null s1)))))

Additional Axioms

(forall (x)
  (iff (compositeEntity x)
       (exists (s1 s2 s3)
         (and (componentsOf s1 x) (propertiesOf s2 x)
              (relationsOf s3 x)))))
