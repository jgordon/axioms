Predicate: quadrilateral

Characterizing Axioms

If n is 4, we have a quadrilateral.

(forall (p)
  (iff (quadrilateral p) (polygon p 4)))

Additional Axioms

A rhombus is a quadrilateral whose sides are all equal.

(forall (p)
  (iff (rhombus p)
       (and (quadrilateral p)
            (exists (d u)
              (forall (c)
                (if (sideOf c p) (length d c u)))))))

Now we can define a rectangle as a quadrilateral whose vertices are all right angles.

(forall (p)
  (iff (rectangle p)
       (and (quadrilateral p)
            (forall (v) (iff (vertexOf v p) (rightAngle v))))))
