Predicate: vertical2Plane

Characterizing Axioms

A vertical2 plane is a set in which no element is rightOf any other. In the two-dimensional case, a vertical2 plane is a vertical2 set.

(forall (s1 f)
   (iff (vertical2Plane s1 f)
        (exists (s)
           (and (domain s f) (subset s1 s) (not (null s1))
                (not (exists (x1 x2)
                        (and (member x1 s1) (member x2 s1)
                             (rightOf x1 x2 f))))))))
