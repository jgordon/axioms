Predicate: lessFct

Additional Axioms

Essentially, x is more functional than y if x is higher on the functionality scale, where being in the Hi region of the functionality scale means something is functional (which means there is an event involving it that brings about its functionality).

(forall (e f s x y)
  (if (and (partialOrdering e x y s) (functionScale s f))
      (lessFct' e x y f)))

The primed version of lessFct. (This would be automatically generated.)

(forall (e f x y)
  (if (and (lessFct' e x y f) (Rexist e))
      (lessFct x y f)))
