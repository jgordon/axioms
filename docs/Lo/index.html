Predicate: Lo

Summary

(Lo s1 s)

s1 is the low region of scale s.

Characterizing Axioms

(forall (s1 s) (if (Lo s1 s) (subscale s1 s)))

(forall (s1 s)
   (if (Lo s1 s)
       (forall (x)
         (iff (bottom x s) (bottom x s1)))))

Additional Axioms

(forall (s s1 s2 x y)
   (if (and (Hi s1 s) (Lo s2 s) (inScale x s1) (inScale y s2))
       (lts y x s)))

near is defined as the Lo region of the nearness scale.

(forall (x z)
   (iff (near x z)
        (exists (s s1)
           (and (nearnessScale s z) (Lo s1 s) (onScale x s1)))))
