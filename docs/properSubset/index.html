Predicate: properSubset

Summary

(properSubset s1 s2)

s1 is a proper subset of s2.

Characterizing Axioms

(forall (s1 s2)
  (iff (properSubset s1 s2)
       (and (subset s1 s2) (nequal s1 s2))))

Additional Axioms

(forall (s1 e2)
  (iff (minimallyProves s1 e2)
       (and (not (member e2 s1))
            (imply s1 e2)
            (not (exists (s2)
                   (and (properSubset s2 s1)
                        (imply s2 e2)))))))
