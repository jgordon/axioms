Predicate: functionInto

Summary

(functionInto f s1 s2)

f is a function from a set or scale s1 into a set or scale s2.

Characterizing Axioms

A function f is "into" a set or scale s3 if there is a subset or subscale s2 for which fi is a function "onto".

(forall (f s1 s3)
   (iff (functionInto f s1 s3)
        (exists (s2)
          (and (or (subset s2 s3) (subscale s2 s3))
               (function f s1 s2)))))

Additional Axioms

A measure is a monotone increasing function from a scale into a non-negative numeric scale in which the bottom of the domain scale, if there is one, maps into 0.

(forall (m s1)
  (iff (measure m s1)
       (exists (s2)
         (and (scale s1) (nonNegNumericScale s2)
              (functionInto m s1 s2)
              (monotoneIncreasing m)
              (forall (x) (if (bottom x s1) (map m x 0)))))))
