Predicate: around

Additional Axioms

If the path of a moving event is a closed curve, the event is “around” the points (and other figures) enclosed by the path.

(forall (e x y1 y2 c)
   (if (and (move' e x y1 y2) (path c e))
       (iff (around c z)
            (exists (r)
               (and (region r) (boundary c r)
                    (inside z r))))))
