Predicate: singleton

Summary

(singleton s x)

s consists of the single element x.

Characterizing Axioms

(forall (s x)
  (iff (singleton s x)
       (exists (s1)
         (and (addElt s s1 x) (null s1)))))

Additional Axioms

(forall (s x y)
  (iff (doubleton s x y)
       (exists (s1)
         (and (nequal x y)
              (addElt s s1 x)
              (singleton s1 y)))))
