Predicate: causalComplex

Summary

(causalComplex c e)

The collection c of eventualities is a causal complex for effect e.

Characterizing Axioms

(forall (s e)
  (if (causalComplex s e)
      (and (eventuality e)
           (or (eventualities s)
               (eventuality s)))))
