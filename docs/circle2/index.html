Predicate: circle2

Characterizing Axioms

A circle is a curve. We use the predicate circle2 to describe the two-dimensional region whose border is a circle.

(forall (r x)
   (iff (circle2 r x)
        (exists (c)
           (and (circle c x) (border c r)))))

Additional Axioms

A closed planar curve has no endpoints and can be covered by a large enough circle.

(forall (c)
   (if (and (curve c)(planar c))
       (iff (closed c)
            (and (not (exists (x) (endpoint x c)))
                 (exists (r x1)
                    (and (circle2 r x1) (subfigure c r)))))))
