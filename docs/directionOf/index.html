Predicate: directionOf

Characterizing Axioms

The direction of a linear object with respect to a framework is a ray that is parallel to the object.

(forall (d c f)
   (iff (directionOf d c f)
        (and (direction d f) (parallel c f))))

Additional Axioms

The orientation of a volume v with a designated face r in a framework f is the direction of a ray c from a central point in v passing through a central point in r, i.e., the direction of a quill.

(forall (d v r f)
   (iff (orientation0 d v r f)
        (exists (c)
           (and (quill c v r) (directionOf d c f)))))
