Predicate: lineSegment

(lineSegment c)

c is a line segment.

Characterizing Axioms

(forall (c)
  (if (lineSegment c)
      (exists (x1 x2)
        (and (endpoint x1 lc (endpoint x2 c)
             (nequal x1 x2)))))


(forall (d c u)
  (iff (lineSegment c)
       (iff (length d c u)
            (exists (x1 x2)
              (and (lineSegmentFromTo c x1 x2)
                   (distance d x1 x2 u))))))

(forall (c)

       (or (line c) (ray c) (lineSegment c))))
