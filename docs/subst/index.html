Predicate: subst

Summary

(subst x e1 y e2)

x plays the same role in e1 that y plays in e2.

Characterizing Axioms

(forall (x e1 y e2)
  (iff (subst x e1 y e2)
       (exists (p n)
         (and (pred p e1) (pred p e2) (arity n e1) (arity n e2)
              (noSubstTypelt x e1 y e2)
              (forall (i z1 z2)
                (if (and (posInteger i) (leq i n)
                         (argn z1 i e1) (argn z2 i e2))
                    (and (iff (eventuality z1)
                              (eventuality z2))
                         (if (not (eventuality z1))
                             (and (if (nequal z1 x)
                                      (equal z2 z1))
                                  (if (equal z1 x)
                                      (equal z2 y))))
                         (if (eventuality z1)
                             (subst x z1 y z2)))))))))
