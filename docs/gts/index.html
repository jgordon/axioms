Predicate: gts

Summary

(gts x y s)

x is greater than y in the partial ordering for scale s.

Characterizing Axioms

(forall (x y s)
  (iff (gts y x s)
       (lts x y s)))

Additional Axioms

(forall (x y s)
   (iff (geqs x y s)
        (or (gts x y s)
            (and (onScale x s) (equal x y)))))
