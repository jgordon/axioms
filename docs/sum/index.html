Predicate: sum

Summary

(sum n1 n2 n3)

n1 is the sum of n2 and n3; n1 = n2+n3.

Characterizing Axioms

(forall (n n1 n2)
   (iff (sum n n1 n2) (sum n n2 n1)))

Additional Axioms

(forall (n s)
  (iff (card n s)
       (and (nonNegInteger n) (set s)
            (or (and (null s) (equal n 0))
                (exists (s1 x m)
                  (and (addElt s s1 x)
                       (not (member x s1))
                       (card m s1)
                       (sum n m 1)))))))
