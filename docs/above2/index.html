Predicate: above2

Additional Axioms

A non-null subset of the domain is “horizontal” if no element is above2 or above3 any other element. If there is no above3 relation, then (above3 x1 x2 f) will always be false, so the definition covers the two-dimensional case as well.

(forall (s1 f)
  (iff (horizontal2 s1 f)
       (exists (s)
         (and (domain s f) (subset s1 s) (not (null s1))
              (not (exists (x1 x2)
                     (and (member x1 s1) (member x2 s1)
                          (or (above2 x1 x2 f)
                              (above3 x1 x2 f)))))))))
