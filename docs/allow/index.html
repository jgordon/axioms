Predicate: allow

Summary

(allow e1 e2)

e1 doesn't cause not e2.

Characterizing Axioms

An eventuality e1 allows an eventuality e2 if e1 does not cause (not e2).

(forall (e1 e2)
  (iff (allow e1 e2)
       (forall (e4)
         (if (not' e4 e2) (not (cause e1 e4))))))
