Predicate: map

Summary

(map f x y)

The function f maps x to y; f(x)=y.

Characterizing Axioms

f.

(forall (f x y)
  (iff (map f x y)
       (exists (s1 s2 p)
         (and (function0 f s1 s2) (member p f)
              (first x p) (second y p)))))

Additional Axioms

(forall (s n y)
  (if (sequence s)
      (iff (nth y n s) (map s n y))))
