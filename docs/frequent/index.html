Predicate: frequent

Summary

(frequent s1)

Set s1 of eventualities is in the high region of a rate scale.

Characterizing Axioms

A set of events is frequent if it is in the Hi region of a rate scale.

(forall (s1)
  (iff (frequent s1)
       (exists (s s0 s2)
         (and (member s1 s0) (rateScale s s0) (Hi s2 s)
              (inScale s1 s2)))))
