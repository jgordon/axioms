Predicate: successor

Summary

(successor n1 n)

n1 is the successor of n; n1 = n+1.

Characterizing Axioms

(forall (n1 n)
   (if (successor n1 n)
       (and (nonNegInteger n1) (nonNegInteger n))))

Additional Axioms

s after the first is removed.

(forall (s1 s)
  (iff (rest s1 s)
       (and (sequence s) (sequence s1)
            (forall (i x)
              (iff (map s1 i x)
                   (exists (j)
                     (and (posInteger i) (successor j i)
                          (map s j x))))))))
