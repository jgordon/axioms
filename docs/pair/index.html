Predicate: pair

Summary

(pair p x y)

p is the ordered pair .

Characterizing Axioms

(forall (p x y)
  (if (pair p x y)
      (and (first x p) (second y p))))

Additional Axioms

If pair0, it's an ordered pair, which has two elements.

(forall (p)
  (iff (pair0 p)
       (exists (x y) (pair p x y))))
