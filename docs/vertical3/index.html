Predicate: vertical3

Characterizing Axioms

A non-null subset of the domain is vertical3 if no element is rightOf or above2 any other element.

(forall (s1 f)
   (iff (vertical3 s1 f)
        (exists (s)
           (and (domain s f) (subset s1 s) (not (null s1))
                (not (exists (x1 x2)
                        (and (member x1 s1) (member x2 s1)
                             (or (rightOf x1 x2 f)
                                 (above2 x1 x2 f)))))))))

Additional Axioms

There may be a unique vertical3 set which is the z-axis, intersecting the x-axis and the y-axis at the origin.

(forall (a3 f)
   (if (zAxis a3 f)
       (and (framework f) (vertical3 a3 f))))

horizontal3 and vertical3 sets are independent scales.

(forall (s1 s2 f)
   (if (and (framework f) (horizontal3 s1 f) (vertical3 s2 f))
       (independentScales s1 s2)))
