Predicate: difficultyScale

Summary

(difficultyScale s e)

s is a scale for measuring the difficulty of actions of type e.

All Axioms

If the set of difficulties associated with achieving e1 contains the set of difficulties associated with achieving e2, then e1 is more difficult than e2.

(forall (s s1 e)
  (if (and (difficultyScale s e) (difficultiesWith' e1 s e))
      (subsetConsistent s e1)))

Something is difficult if it is in the Hi region of a difficulty scale.

(forall (e1 e a)
  (iff (difficult' e1 e a)
       (and (agent a) (agentOf a e)
            (exists (s)
              (and (difficultyScale s e) (scaleFor s e1))))))
