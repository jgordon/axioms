Predicate: polygon

Characterizing Axioms

(forall (p n)
  (iff (polygon p n)
       (exists (s)
         (and (forall (c)
                (iff (member c s) (sideOf c p)))
              (forall (x)
                (if (inside x p)
                    (exists (c)
                      (and (member c s)
                        (or (inside x c) (endpoint x l))))))
              (closed p) (planar p)
              (card n s)))))
