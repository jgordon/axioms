Predicate: body

(body b p)

b is p's body.

Characterizing Axioms

(forall (b p)
  (if (body b p)
      (physobj b)))


(forall (b p)
  (if (body b p)
      (and (spatialSystem b)
           (exists (s)
             (and (componentsOf s b)
                  (forall (x)
                    (iff (member x s) (bodyPart x p))))))))
